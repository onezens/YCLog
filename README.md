# YCLog

## deviceconsole
[https://github.com/MegaCookie/deviceconsole](https://github.com/MegaCookie/deviceconsole)

**clone deviceconsole**

```
git clone https://github.com/MegaCookie/deviceconsole
```
**complie**

```
cd deviceconsole
make
```
**set as global**

```
sudo cp ./deviceconsole /usr/local/bin
```
